Простой API для работы с заявками на займ, реализован на Yii 2 (PHP 8.3 + Docker).

---

## Функционал

1. **Создание новой заявки**
   - Эндпоинт: `POST /requests`
   - Проверка: пользователь **не может иметь одобренные заявки**.
   - Каждая новая заявка создаётся со статусом `pending`.
   - Пример успешного ответа:
     ```json
     {
       "result": true,
       "id": 42
     }
     ```
   - Пример неуспешного ответа:
     ```json
     {
       "result": false
     }
     ```

2. **Обработка заявок**
   - Эндпоинт: `GET /processor?delay=5`
   - Все `pending` заявки обрабатываются:
      - Статус определяется случайно: вероятность `approved` — 10%, `declined` — 90%.
      - У одного пользователя не может быть более одной одобренной заявки.
   - `delay` эмулирует время принятия решения через `sleep()`.
   - Параллельная обработка: несколько вызовов эндпоинта можно запускать одновременно, заявки одного пользователя обрабатываются параллельно.
   - Пример ответа:
     ```json
     {
       "result": true
     }
     ```

## Поднятие проекта через Docker

1. **Сборка контейнеров:**
```bash
docker compose build
docker compose up -d
 ```
2. **Установка зависимостей**
```bash
docker compose run --rm app composer install
 ```
3. **Применение миграций:**
```bash
docker compose run --rm app php yii migrate
 ```

4. **Наполнение таблицы пользователей (тестовые данные):**
```bash
docker compose run --rm app php yii seed/users
 ```

5. **Доступ к приложению**
```bash
http://localhost
 ```

## Конфигурация

1. **Параметры задержки обработки заявок: config/params.php**
```php
return [
    'loanProcessingDelay' => 5, // задержка обработки заявок в секундах
];
 ```

2. **База данных: config/db.php**
```php
return [
'class' => 'yii\db\Connection',
'dsn' => 'pgsql:host=db;port=5432;dbname=loans',
'username' => 'user',
'password' => 'password',
'charset' => 'utf8',
];
 ```
## Тестирование эндпоинтов

1. **Создание заявки:**
```bash
curl -X POST http://localhost/requests 
-H "Content-Type: application/json" 
-d '{"user_id":1,"amount":3000,"term":30}'
 ```
2. **Обработка заявок:**
```bash
curl -X GET "http://localhost/processor?delay=1"
 ```

## Что реализовано

- Создана команда, которая автоматически создаст 10 тестовых пользователей в таблице users.
  - Позволяет сразу тестировать создание и обработку заявок без ручного добавления пользователей.
- Создание новых заявок (pending) с проверкой на существующие approved заявки.
- Эндпоинт обработки заявок:
  - Рандомное принятие решения (approved/declined) с вероятностью 10% для approved.
  - Возможность многократного запуска параллельно.
  - Обработка заявок одного пользователя параллельно.
- Простая JSON-обёртка для всех ответов:
  - result: true/false
  - id для новых заявок
  - Сообщение об ошибке (при необходимости)
- Полностью тестируемо через cURL или Postman.
